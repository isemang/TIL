## 작성 규칙

- 파일명 : [20XX.XX.XX] "폴더명(오늘 공부한 항목 상위 분류)" - "제목(공부한 내용 상세)"

- 작성 후 반드시 TIL Wiki에 해당 내용을 업데이트한다.
- 다양한 내용을 공부했다면, 하나에 몰아 작성하지 말고 나눠서 각각 작성한다.
